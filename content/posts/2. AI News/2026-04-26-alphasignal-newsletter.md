### AI, 이제 '알아서' 처리합니다: 에이전트 시대의 도래

OpenAI는 GPT-5.5 출시와 워크스페이스 에이전트 도입을 통해 '프롬프트 시대의 종말'을 선언했습니다. 이제 AI는 단순히 질문에 답하는 것을 넘어, 복잡한 다단계 작업을 스스로 계획하고, 도구를 사용하며, 작업을 확인하고, 완료하는 '에이전트'로 진화하고 있습니다.

*   **OpenAI GPT-5.5 출시**: 에이전트 코딩 능력과 성능이 크게 향상되어, AI에게 복잡한 작업을 맡기면 스스로 처리합니다. Terminal-Bench 2.0에서 82.7%, SWE-Bench Pro에서 58.6%의 높은 점수를 기록했으며, API 가격은 백만 토큰당 5달러부터 시작하고 100만 토큰 컨텍스트를 지원합니다.
*   **OpenAI 워크스페이스 에이전트 출시**: ChatGPT 내에서 팀 워크플로우를 자동화하는 에이전트가 도입되었습니다. 소프트웨어 요청 분류, 정책 시행, 승인 라우팅, Slack 피드백 수집 및 제품 액션 전환 등 다양한 업무를 자동화하여 팀 생산성을 극대화할 수 있습니다. 2026년 5월 6일까지 무료로 제공됩니다.
*   **MIT 1000만 토큰 재귀 모델 프레임워크**: MIT 연구진은 1000만 토큰을 처리하며 표준 모델보다 58배 우수한 성능을 보이는 재귀 모델 프레임워크를 공개했습니다. 이는 컨텍스트 길이의 병목 현상이 더 이상 문제가 아님을 보여주며, AI가 더 광범위한 정보를 이해하고 처리할 수 있게 됨을 의미합니다.
*   **OpenAI 멀티 에이전트 워크플로우용 Python SDK 오픈소스화**: OpenAI는 복잡한 에이전트 시스템 구축을 위한 오픈소스 Python SDK를 출시하여 개발자들이 멀티 에이전트 워크플로우를 보다 쉽게 만들 수 있도록 지원합니다.

### 오픈소스 LLM의 놀라운 진화: 더 작고 강력하게

오픈소스 LLM 분야에서는 효율성과 성능 면에서 놀라운 발전이 이어지고 있습니다. 더 작고 접근하기 쉬운 모델들이 기존의 대규모 모델들을 능가하는 사례가 등장하며, 기술 혁신을 주도하고 있습니다.

*   **Qwen 오픈소스 27B 모델, 397B 모델 능가**: Alibaba의 새로운 오픈소스 모델 Qwen3.6-27B는 27B라는 작은 규모에도 불구하고, 자체 397B 모델을 코딩 성능에서 뛰어넘었습니다. SWE-bench Verified에서 77.2점, Terminal-Bench 2.0에서 59.3점을 기록했으며, 약 18GB VRAM으로 로컬 실행이 가능하고 Apache 2.0 라이선스를 따릅니다. 'Thinking Preservation' 기능으로 추론 과정을 기억하여 다단계 코딩 작업을 효율적으로 수행합니다.
*   **Unsloth의 양자화된 Qwen3.6-27B**: Unsloth는 미세 조정 및 도구 호출을 지원하는 양자화된 Qwen3.6-27B를 출시하여, 이 강력한 오픈소스 모델의 활용성을 더욱 높였습니다.
*   **상수 메모리 재귀 모델 (텍스트 임베딩)**: 트랜스포머의 한계를 극복하고 상수 메모리로 텍스트 임베딩을 생성하는 새로운 재귀 모델이 등장했습니다. 이는 효율적인 AI 모델 개발에 중요한 기술적 진보입니다.
*   **LLM의 직접 해결 불가 문제 추론 학습 미세 조정 방법**: LLM이 직접 해결할 수 없는 문제에 대해 추론하도록 가르치는 새로운 미세 조정 방법이 발표되어, LLM의 문제 해결 능력 향상에 기여할 것으로 기대됩니다.

### AI 활용의 문턱을 낮추다: 접근성과 효율성

AI 기술의 발전과 함께, 이를 더 저렴하고 효율적으로 활용할 수 있는 방안들도 활발히 모색되고 있습니다.

*   **free-claude-code: Claude Code 무료 사용**: 오픈소스 프로젝트 'free-claude-code'는 Claude Code의 API 호출을 NVIDIA NIM(분당 40회 요청 무료) 또는 로컬 환경으로 라우팅하여, 강력한 AI 코딩 도구를 무료로 사용할 수 있게 합니다.
*   **2025년 1000달러 GPU로 로컬 실행 가능한 오픈웨이트 모델**: 2025년에는 1000달러 수준의 GPU로도 로컬에서 강력한 오픈웨이트 모델을 실행할 수 있게 될 것이라는 전망이 나왔습니다. 이는 개인 개발자나 소규모 팀에게 AI 접근성을 크게 높여줄 것입니다.
*   **문서를 PowerPoint로 변환하는 오픈소스 도구**: 어떤 문서든 완전히 편집 가능한 PowerPoint 덱으로 변환해주는 오픈소스 도구가 공개되어, 문서 작업의 효율성을 높일 수 있습니다.

### AI의 새로운 지평과 심층적 고찰

AI 기술은 다양한 분야로 확장되며 새로운 가능성을 열고 있지만, 동시에 그 활용에 대한 심층적인 고찰도 필요합니다.

*   **GPT Image 2 기반 3D 메시 텍스트 편집 (CADAM)**: GPT Image 2를 활용한 오픈소스 브라우저 앱 CADAM은 텍스트 프롬프트만으로 3D CAD 모델을 생성하고 편집할 수 있게 합니다. 이는 3D 디자인 분야에서 AI의 활용 가능성을 보여주는 혁신적인 사례입니다.
*   **LLM Fallacy: AI 의존에 따른 실제 능력 저하 경고**: 새로운 연구는 AI가 좋은 결과물을 내면 사용자가 자신의 능력으로 착각하여 실제 기술이 퇴화하는 'LLM Fallacy' 현상을 경고합니다. 코딩, 글쓰기, 분석, 언어 학습 등 다양한 분야에서 AI에 대한 과도한 의존이 실제 능력 저하로 이어질 수 있음을 인지해야 합니다.
*   **Sakana AI의 LLM 무작위/다양한 출력 생성 프롬프팅 기술**: Sakana AI는 LLM이 더 무작위적이고 다양한 출력을 생성하도록 하는 프롬프팅 기술을 선보였습니다. 이는 AI의 창의성과 유연성을 높이는 데 기여할 수 있습니다.
